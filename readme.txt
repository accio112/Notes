This is a simple yet useful app to add, read, delete and list the notes.

Notes can be added by typing the below command in console:
node app.js --title="title of the note" --body="body of the note"

Notes can be listed by
node app.js list

Notes can be removed by providing the title of the node to be deleted.
node app.js remove --tilte="title of the note"

Notes can be read by the command
node app.js read --title="title of the node"
